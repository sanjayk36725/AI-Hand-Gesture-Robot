# AI Hand Gesture Robot Controller 🤖

This project uses AI (MediaPipe) to detect hand gestures and control a virtual robot.

## Features
- Hand gesture recognition
- Real-time robot movement
- Uses MediaPipe AI

## Controls
- Open Palm → STOP
- Fist → FORWARD
- Point Left → LEFT
- Point Right → RIGHT

## Tech Stack
- HTML, CSS, JavaScript
- MediaPipe AI
